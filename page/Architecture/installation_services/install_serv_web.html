Installation serveur web

1. Installer apache

Nous utiliserons Apache comme server HTTP. C’est une des technologies les plus utilisé dans le domaine de l’hébergement de serveur web.
Apache est disponible par défaut dans le répertoire d’application d’Ubuntu. Ainsi pour télécharger et installer la dernière version rentrez les commandes suivantes
Sudo apt update
Sudo apt install apach2
Après avoir confirmé l’installation, apt installera automatiquement apache ainsi que toutes ses dépendances.

2. Ajuster le firewall de votre machine

Une fois apache installer, il est nécessaire d’ajuster le parefeu intégré à ubuntu (UFW) afin d’autoriser les connection externe au port http (et https) par defaut
Commençons par afficher la liste des application enregistrés auprès d’UFW :

Sudo ufw app list

Apache correspond au port 80, Apache Secure au port 443 et Appach Full au deux.

Dans notre cas nous ouvrirons l’accès au port 80 : 

Sudo ufw allow ‘Apach Full’
Nous pouvons vérifier le résultat : 
Sudo ufw status


3. Tester votre serveur

Notre serveur apache devrait désormais être opérationnelle.
Nous allons vérifier cela en deux étapes. La première est de vérifier l’état du service : 

Sudo systemctl status apache2

Ensuite pour confirmer que le logiciel s’est correctement installé nous allons accéder à la page par défaut mise en place par apache.
Pour cela il suffit de rentrer l’ip sur serveur web sur votre navigateur (dans le réseau local).
