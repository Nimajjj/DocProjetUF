Configuration réseau d’un serveur

Les serveurs ont besoin d’un IP statique et toutes les machines linux ont besoin de définir leur passerelle par défaut.
Pour se faire nous vous invitons à ouvrir les paramètres de votre serveur :

Dans l’onglet « Network » (cf. 1) rentrer dans les paramètres de votre connexion (cf. 2).


Puis dans l’onglet addresses : rentrer l’ip statique que vous souhaitez pour votre serveur (cf. a), le masque de sous réseau définis au préalable (cf. b) et enfin l’ip de votre pare-feu vous reliant à internet (cf. c)
